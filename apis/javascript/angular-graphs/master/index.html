Angular 2 Graphs example

This is a demo of an Typescript implementation of a graph model and different graph algoritms. It is based on my previous examples in Dart examples repository. Currently the feature set is not complete:

  • nodes can be created and dragged around

The graph model and UI logic is implemented in pure TypeScript. The model and UI is then wired into Angular 2 API, so it can be easily used in an Angular app.
